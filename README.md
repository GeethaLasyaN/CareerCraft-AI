# CareerCraft AI

CareerCraft AI is a futuristic AI-powered career development platform built using React and Tailwind CSS.

## Features

- AI-inspired Resume Analyzer
- Interview Preparation Assistant
- LinkedIn Optimization Experience
- Interactive Career Assistant Chatbot
- Modern futuristic UI/UX
- Responsive design

## Tech Stack

- React.js
- Tailwind CSS
- Vite
- JavaScript

## Purpose

This project explores how Generative AI-inspired systems and modern frontend experiences can improve career guidance and user interaction.
